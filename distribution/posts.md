## 5. Hacker News — Show HN

**Title:** Show HN: War Room – Single HTML file live dashboard for the Iran-USA-Israel conflict

**Body:**

I built a real-time war room dashboard tracking the Iran-USA-Israel conflict as a single self-contained HTML file. No build step, no backend, no npm. Just one index.html deployed to Vercel.

Live at: https://war-room.live

**Technical decisions:**

- **Single file architecture (~2,000 lines):** The entire dashboard — CSS, HTML, JS — lives in one index.html. No bundler, no framework. This was deliberate. When you're the only person updating a live crisis dashboard, deployment needs to be `git push` and done. Vercel auto-deploys in ~30 seconds.

- **Zero dependencies (CDN only):** Leaflet.js for maps, Google Fonts for typography, CartoDB dark tiles for basemaps. Everything else is vanilla JS. The market tickers, event log, escalation scale, i18n system — all hand-written.

- **i18n without a library:** 8 languages (EN, ES, FR, PT, RU, AR, ZH, FA) implemented with a `data-i18n` attribute system and a `LANGS{}` object. Auto RTL switching for Arabic and Farsi via a body class. No i18next, no react-intl. About 200 lines of JS.

- **Claude API integration:** The dashboard has a built-in AI analyst chat panel. The system prompt contains the full conflict context — every verified event, market figure, and escalation indicator. Users provide their own Anthropic API key. The AI can answer analytical questions with full situational awareness. API calls go directly to Anthropic's API from the browser — no proxy server needed.

- **Leaflet map with custom markers:** Pulsing red circles for strike targets (CSS animation), pentagon shapes for bases, triangles for naval assets. Two separate map instances — one mini-map for the theater, one world map for global force positions. Both support theme switching (dark/light tiles).

- **Market flicker effect:** The market strip simulates live-updating prices with JS intervals that add small random deltas to base values. Gives the feel of a Bloomberg terminal without requiring a data feed websocket.

- **CSS-only features:** Dark/light theme via CSS custom properties on :root. Responsive 3-column layout that collapses on mobile. Kahn escalation scale is pure CSS with a highlighted active segment. Collapsible map with CSS transitions.

- **Dual floating panels:** X world positions panel and AI chat panel are floating FABs that expand into full panels. CSS positioning, no dialog/modal library.

**What it tracks:**

- Verified event timeline (D+0 through D+5) from 20+ sources
- 10 market indicators: Brent, WTI, Gold, S&P, Dow, 10Y, VIX, DXY, VLCC, EU Gas
- Kahn Escalation Scale (currently Level 8)
- 5 intelligence analysis frameworks (PMESII-PT, Red/Blue Team, COA, Balance of Power, Scenarios)
- Conflict maps with strike locations, bases, naval positions
- FX rates and DXY gauge

Built over a weekend when the conflict started and I couldn't find a single dashboard that combined verified events with market data and structured analysis. Happy to discuss the architecture choices and trade-offs of the single-file approach.
